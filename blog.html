Question-1: What are the different ways to select an element in the DOM?

  • getElementById(): একটি নির্দিষ্ট আইডি সহ উপাদান নির্বাচন করে।
  • getElementsByClassName(): একটি নির্দিষ্ট ক্লাস নাম সহ সমস্ত উপাদান নির্বাচন করে।
  • getElementsByTagName(): একটি নির্দিষ্ট ট্যাগ নাম সহ সমস্ত উপাদান নির্বাচন করে।
  • querySelector(): CSS সিলেক্টর ব্যবহার করে প্রথম মিলে যাওয়া উপাদান নির্বাচন করে।
  • querySelectorAll(): CSS সিলেক্টর ব্যবহার করে সমস্ত মিলে যাওয়া উপাদান নির্বাচন করে।

Question-2: What is the difference between innerHTML, innerText, and textContent ?

  • innerHTML: এটি একটি উপাদানের ভিতরে থাকা HTML সেট করতে ব্যবহৃত হয়।
  • innerText: এটি একটি উপাদানের দৃশ্যমান টেক্সট (যা ব্যবহারকারী দেখতে পারে) সেট করতে ব্যবহৃত হয়।
  • textContent: এটি একটি উপাদানের সমস্ত টেক্সট কন্টেন্ট (দৃশ্যমান এবং গোপন টেক্সট সহ) সেট করতে ব্যবহৃত হয়।

Question-3: What is event delegation in the DOM?

ইভেন্ট ডেলিগেশন একটি টেকনিক যেখানে একটি parent element এ একটি একক event listener যুক্ত করা হয়, যা তার সমস্ত child element এর ইভেন্টগুলো ম্যানেজ করে। এটি event propagation এবং event.target ব্যবহার করে কাজ করে, যা ইভেন্টটির মূল উৎস (target) চিহ্নিত করতে সাহায্য করে।

Question-4: What is event bubbling in the DOM?

ইভেন্ট বুবলিং একটি প্রক্রিয়া যেখানে একটি ইভেন্ট প্রথমে target element এ শুরু হয় এবং তারপর এটি তার parent element গুলোর মধ্য দিয়ে উপরের দিকে bubble করতে থাকে। এর মাধ্যমে, parent element গুলি তাদের child element এ ঘটে যাওয়া ইভেন্টগুলি handle করতে পারে।

Question-5: How do you create, add, and remove elements using JavaScript?

    /* Create an element */
    const newEl = document.createElement('div');
    newEl.textContent = "New Element";
    
    /* Add the element to the DOM */
    document.body.appendChild(newEl);
    
    /* Remove the element from the DOM */
    newEl.remove();

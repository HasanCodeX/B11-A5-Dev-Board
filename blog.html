1. What are the different ways to select an element in the DOM?

  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()

2. What is the difference between innerHTML, innerText, and textContent?

  • innerHTML: Gets or sets HTML markup contained within the element.
  • innerText: Gets or sets the visible text (ignores hidden elements).
  • textContent: Gets or sets the full text content, including hidden elements.

3. What is event delegation in the DOM?

Event delegation is a technique where a single event listener is attached to a parent element to manage events from multiple child elements. It works by utilizing the event propagation and event.target to determine which child element triggered the event.

4. What is event bubbling in the DOM?

Event bubbling is a process where an event starts at the target element and then bubbles up through its ancestors in the DOM hierarchy. This allows parent elements to handle events that occur on their children.

5. How do you create, add, and remove elements using JavaScript?

/* Create an element */
const newEl = document.createElement('div');
newEl.textContent = "New Element";

/* Add the element to the DOM */
document.body.appendChild(newEl);

/* Remove the element from the DOM */
newEl.remove();
